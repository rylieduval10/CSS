Step 2: Custom Dropdowns (Optional)

A. Add JavaScript Files

If you want to use the custom dropdowns, add these script tags to your page (right before closing </body> tag):

<script src="custom-dropdown.js"></script>
<script src="position-badges.js"></script>

B. Update HTML Structure

Replace your old <select> dropdowns with this structure:

<div class="modern-filter-group">
    <div class="custom-select" data-name="playerFilter">
        <div class="custom-select-trigger">
            <span class="custom-select-value">Only Top Players</span>
            <span class="custom-select-arrow"></span>
        </div>
        <div class="custom-select-options">
            <div class="custom-select-option selected" data-value="top">Only Top Players</div>
            <div class="custom-select-option" data-value="all">All Players</div>
            <div class="custom-select-option" data-value="rookies">Rookies Only</div>
        </div>
        <select name="playerFilter">
            <option value="top" selected>Only Top Players</option>
            <option value="all">All Players</option>
            <option value="rookies">Rookies Only</option>
        </select>
    </div>
</div>
How it works: The hidden <select> element still exists for form submission. The JavaScript automatically syncs the custom dropdown with the hidden select, so your form posts work exactly the same.

Step 3: Keep Selections After Refresh (Optional)

The Problem:

When a user selects "2B" and clicks "Refresh", the page reloads and 2B is no longer highlighted (even though the data is filtered correctly).

The Solution:

Add server-side code to check what was posted and restore the selections.

A. Add Helper Functions (in your .aspx.cs or inline):

protected bool IsPositionSelected(string position)
{
    string[] selectedPositions = Request.Form.GetValues("positions[]");
    return selectedPositions != null && selectedPositions.Contains(position);
}

protected string GetActiveClass(string position)
{
    return IsPositionSelected(position) ? "active" : "";
}

protected string GetChecked(string position)
{
    return IsPositionSelected(position) ? "checked" : "";
}

B. Update Position Badge HTML:

<label class="modern-filter-badge <%= GetActiveClass("2B") %>" data-pos="2B">
    <input type="checkbox" name="positions[]" value="2B" <%= GetChecked("2B") %>>
    2B
</label>

C. Update "All" Badge:

<div class="modern-filter-badge <%= Request.Form.GetValues("positions[]") == null || 
     Request.Form.GetValues("positions[]").Length == 0 ? "active" : "" %>" 
     id="all-badge">All</div>

D. Complete Working Example:

Here's a complete example showing all the pieces together:

<%@ Page Language="C#" %>

<script runat="server">
    protected bool IsPositionSelected(string position)
    {
        string[] selectedPositions = Request.Form.GetValues("positions[]");
        return selectedPositions != null && selectedPositions.Contains(position);
    }
    
    protected string GetActiveClass(string position)
    {
        return IsPositionSelected(position) ? "active" : "";
    }
    
    protected string GetChecked(string position)
    {
        return IsPositionSelected(position) ? "checked" : "";
    }
</script>

<form method="POST">
    <div class="modern-filter-badges">
        <div class="modern-filter-badge <%= Request.Form.GetValues("positions[]") == null ? "active" : "" %>" id="all-badge">All</div>
        <div class="modern-filter-badge" id="clear-badge">Clear</div>
        
        <label class="modern-filter-badge <%= GetActiveClass("C") %>" data-pos="C">
            <input type="checkbox" name="positions[]" value="C" <%= GetChecked("C") %>>
            C
        </label>
        
        <label class="modern-filter-badge <%= GetActiveClass("1B") %>" data-pos="1B">
            <input type="checkbox" name="positions[]" value="1B" <%= GetChecked("1B") %>>
            1B
        </label>
        
        <!-- Add remaining positions... -->
    </div>
    
    <button type="submit">Refresh</button>
</form>

<script src="position-badges.js"></script>

Step 4: Customizing Position Colors

To Change Any Position Color:

Edit the sport-specific CSS file (bm_mlb_light_2025-01-26_08.css or bm_mlb_dark_2025-01-26_08.css) and find the :root section:

:root {
    --pos-c: #0099FF;      /* Cyan - Catcher */
    --pos-1b: #339900;     /* Green - 1st Base */
    --pos-2b: #0000cc;     /* Blue - 2nd Base */
    --pos-ss: #cc0000;     /* Red - Shortstop */
    --pos-3b: #ff6600;     /* Orange - 3rd Base */
    --pos-of: #660099;     /* Purple - Outfield */
    --pos-dh: #1f2937;     /* Dark gray - DH */
    --pos-sp: #683300;     /* Brown - Starting Pitcher */
    --pos-rp: #ff0099;     /* Pink - Relief Pitcher */
    --pos-cl: #ff0099;     /* Pink - Closer */
    --pos-ph: #6b7280;     /* Gray - Pinch Hitter */
}

Just replace the hex code with your preferred color!
